# Tmux Configuration

Terminal multiplexer configuration, managed declaratively by Nix Home Manager.

## Overview

- **Custom prefix:** `Ctrl+a` (easier to reach than `Ctrl+b`)
- **Vim-style navigation:** seamless movement between tmux and Neovim (`C-h/j/k/l`)
- **Session persistence:** auto-save / restore (resurrect + continuum)
- **Popup windows:** quick shell, lazygit, fzf session switcher
- **Catppuccin Mocha** theme (v2 status modules)
- **Per-pane git status** via gitmux (great for worktrees)
- **DevPod integration:** Tailscale-based remote sessions (optional)

## How It's Managed

This config is **not** managed by TPM and **not** by symlinking files here. It
is generated by Home Manager from
`modules/home-manager/programs/tmux.nix`, which:

- writes the live config to `~/.config/tmux/tmux.conf`, and
- installs plugins from the Nix store (no `~/.tmux/plugins`, no `tpm`).

> The `tmux.conf` in this directory is a **reference stub** ‚Äî editing it does
> nothing. See `tmux/CLAUDE.md` for the architecture.

### Changing the config

```bash
# 1. Edit modules/home-manager/programs/tmux.nix
# 2. Apply:
home-manager switch --flake .
# 3. Reload the running session:
#    prefix + r
```

There is **no** `prefix + I` plugin-install step ‚Äî plugins are added in
`tmux.nix` and installed on `home-manager switch`.

## Essential Keybindings

**Prefix:** `Ctrl+a`

| Keybinding | Action |
|------------|--------|
| `Prefix + ?` | Show all keybindings |
| `Prefix + r` | Reload configuration |
| `Prefix + d` | Detach from session |

Full list: `docs/KEYBINDINGS_TMUX.md`.

## Core Features

### Windows

- `Prefix + c` ‚Äî new window
- `Prefix + ,` ‚Äî rename window
- `Prefix + 0-9` / `n` / `p` ‚Äî navigate windows
- `Prefix + w` ‚Äî window tree (with pane counts)
- `Prefix + W` ‚Äî full tree (all sessions/windows/panes)

### Panes

**Split:**
- `Prefix + |` ‚Äî split horizontally (opens in current path)
- `Prefix + -` ‚Äî split vertically (opens in current path)

**Navigate (no prefix ‚Äî vim-tmux-navigator):**
- `Ctrl+h/j/k/l` ‚Äî left/down/up/right (works across Neovim splits too)

**Resize (repeatable):**
- `Prefix + H/J/K/L`

**Other:**
- `Prefix + t` ‚Äî rename pane title
- `Prefix + z` ‚Äî zoom pane

### Sessions

- `Prefix + m` ‚Äî new session prompt (in current path)
- `` Prefix + ` `` ‚Äî fzf session switcher (popup)
- `Prefix + X` ‚Äî kill a session via fzf
- `Prefix + d` ‚Äî detach
- `tmux ls` / `tmux attach -t <name>` ‚Äî from the terminal

### Popups

- `Prefix + !` ‚Äî quick shell (60% √ó 75%)
- `` Prefix + ` `` ‚Äî fzf session switcher (60% √ó 50%)
- `Prefix + g` ‚Äî lazygit (80% √ó 80%)

Popups use `display-popup -E` and close when the command exits.

### Copy Mode (vi-style)

- `Prefix + Enter` ‚Äî enter copy mode (`Prefix + [` also works, tmux default)
- `v` ‚Äî begin selection, `V` ‚Äî select line, `Ctrl+v` ‚Äî rectangle
- `y` ‚Äî copy to clipboard (macOS `pbcopy`; tmux-yank handles Linux)
- `/` `?` `n` `N` ‚Äî search
- `q` / `Esc` ‚Äî exit

### Session Persistence

- Auto-save every 60 min, auto-restore on tmux start (continuum)
- Manual: `Prefix + Ctrl+s` save, `Prefix + Ctrl+r` restore
- SSH sessions (DevPod) preserved via `@resurrect-processes 'ssh'`
- State: `~/.tmux/resurrect/`

## Plugins

Installed from the Nix store via `tmux.nix` (`programs.tmux.plugins`):

| Plugin | Purpose |
|--------|---------|
| sensible | sane defaults |
| catppuccin | Mocha theme + status-line modules (v2) |
| resurrect + continuum | session save/restore |
| battery, cpu | status indicators |
| vim-tmux-navigator | seamless Neovim/tmux navigation |
| yank | clipboard integration |
| tmux-fzf | fuzzy finder for sessions/windows/panes |

To add one: append it to the `plugins` list in `tmux.nix` and run
`home-manager switch --flake .`.

## Per-Pane Git Status (gitmux)

Each pane border shows its own git branch + status via gitmux ‚Äî so in
multi-worktree setups you always see which branch a pane is on. Configured in
`tmux/gitmux.conf` (Catppuccin Mocha palette), symlinked to `~/.gitmux.conf`.

Symbols: `‚éá branch` ¬∑ `‚Üën` ahead ¬∑ `‚Üìn` behind ¬∑ `‚óèn` staged ¬∑ `‚úön` modified ¬∑
`‚Ä¶n` untracked ¬∑ `‚öën` stashed ¬∑ `‚úñn` conflicts ¬∑ `‚úî` clean.

## DevPod Integration (optional)

Enabled with `dev-config.tmux.devpodConnect.enable = true`. Tailscale-based:

- `Prefix + D` ‚Äî fzf picker of online DevPods ‚Üí SSH session
- On tmux start, online pods get `devpod_<project>` sessions auto-created
- Mutagen file sync auto-starts (via the `session-created` hook) when the
  project has a `mutagen.yml`

See `tmux/CLAUDE.md` for the script architecture.

## Configuration Tips

All edits go in `modules/home-manager/programs/tmux.nix`, then
`home-manager switch --flake .`.

**Change theme flavor** ‚Äî in the catppuccin plugin attrset:
```nix
set -g @catppuccin_flavor 'latte'  # latte | frappe | macchiato | mocha
```
(Note the spelling: `@catppuccin_flavor`, no `u`.)

**Change prefix** ‚Äî `programs.tmux.prefix = "C-b";`

**Add a keybinding** ‚Äî add a `bind ...` line in `programs.tmux.extraConfig`.

## Troubleshooting

| Symptom | Fix |
|---------|-----|
| Edits to this `tmux.conf` ignored | Expected ‚Äî it's a stub. Edit `tmux.nix`. |
| Theme not applying | Ensure `@catppuccin_flavor` (no `u`) is in the catppuccin plugin attrset. |
| Colors wrong | `echo $TERM` ‚Üí should be `tmux-256color`. |
| Navigation broken | Confirm Neovim has matching `C-h/j/k/l` binds; `tmux kill-server && tmux`. |
| Session not restoring | `ls ~/.tmux/resurrect/`; `Prefix + Ctrl+r`. |
| Git status blank | `which gitmux`; check `~/.gitmux.conf`; `gitmux -cfg ~/.gitmux.conf $(pwd)`. |

## Tips & Tricks

```bash
tmux rename-session new-name      # or: Prefix + $
tmux kill-session -a              # kill all but current
:setw synchronize-panes on        # type into all panes at once
:swap-window -t 1                 # reorder windows
```
